Gravity Game

About

This game was a small project of mine who wanted to see if it was posible to make a gravity system without much help.

RailWays

This is the code used for calculating gravity. First, it calculates the distance to each planet, and then it calculates the force (gravity) that needs to be added towards the planet. Then it adds the force to the bullet, and it moves forward based on its current velocity.
